# Tarifaço do Governo Trump II e Derrubada das Tarifas – Seção 122

## Descrição
Este projeto analisa as medidas tarifárias adotadas pelos Estados Unidos
durante o governo Trump II, incluindo alterações relacionadas à Seção 122
e revisões no Harmonized Tariff Schedule (HTS).

## Dashboard Interativo
O painel interativo desenvolvido para este projeto pode ser acessado em:

🔗 https://tarifapanelelisangelagelatti.manus.space/

## Fonte de Dados e Metodologia
Este projeto foi desenvolvido com base em dados de comércio internacional dos Estados Unidos,
obtidos a partir do sistema USA Trade Online do U.S. Census Bureau.

 🔗 https://usatrade.census.gov/index.php

A análise incorpora as medidas tarifárias adotadas pelo governo Trump II,
incluindo Ordens Executivas (Executive Orders) e alterações tarifárias
no Harmonized Tariff Schedule (HTS). As revisões tarifárias foram verificadas no sistema oficial do HTS:

🔗 https://hts.usitc.gov/
